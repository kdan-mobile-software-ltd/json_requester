# Security Policy

## Supported Versions

The following versions are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| 2.x     | Yes       |
| 1.x     | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in `json_requester`, please report it privately by email to `dev@kdanmobile.com`.

Please do not open a public issue for security-related reports.

When reporting a vulnerability, please include the following information whenever possible:

- A clear description of the issue
- The affected version(s)
- Steps to reproduce the issue
- The potential impact
- Any suggested remediation, if available

## Response Process

We will make a reasonable effort to:

- Acknowledge receipt of your report within 5 business days
- Investigate and validate the issue
- Share next steps or a remediation timeline when possible

## Disclosure Policy

Please allow time for the issue to be investigated and fixed before disclosing it publicly. Coordinated disclosure helps protect all users of the project.
